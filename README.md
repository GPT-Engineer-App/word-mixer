# word-mixer

A website where you put in a word and it will randomly put a word from a word dictionary after, you can write the dictionary or use a dictionary from the internet that you fetch via API

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/word-mixer.git
cd word-mixer
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
